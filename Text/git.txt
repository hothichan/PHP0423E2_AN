Kỹ năng cơ bản về Git
1 - Git là gì, tại  sao cần Git?
	+ Là công cụ quản lý source code (giống Google Drive) và code nhóm hiệu quả
	Bài toán: nếu không có Git, gộp code giữa các thành viên sẽ bị thủ công -> sai sót
	+ Có nhiều dịch vụ cung cấp sẵn Git: GitHub, Gitlap, Bitbucket,...
2 - Demo quản lý source code với Git:
	+ Đăng ký tài khoản
	+ git init -> khởi taoh trên git trên thư mục hiện tại
	+ git remote -v -> kiểm tra link repo
	+ git status -> kiểm tra trạn thái file/ thư mục
	+ git add -> thêm file cho git để chuẩn bị cho commit
