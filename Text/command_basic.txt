Một số command line cơ bản:

+ Mở công cụ command line

D: 		-> chuyển sang ổ mới
cd \ 	-> chuyển về ổ gốc
		-> bổ sung kĩ năng đọc hiểu tiếng anh
		cài extension cho chrome: Laban, Google Translate
mkdir <tên thư mục> -> tạo thư mục
echo "nội dung file" > <tên file> -> tạo file
ls -l -> hiển thị danh sách file trong thư mục
cat <tên file> -> đọc nội dung file
cd.. ->trở về thư mục trước
